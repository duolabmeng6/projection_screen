# 多多投屏 永久免费 开源软件

轻轻松松在MacOS和Window中将视频文件投屏到电视上,跟手机的投屏功能一致,无需nas等繁琐操作.直接文件投屏.

# 运行环境

* Window10
* MacOS

电视中需要安装接收端例如

* 当贝投屏 无广告
* ~乐播投屏~ 投屏后有广告恶心人
* 银河奇异果 无广告
